🚀 Minimalist & High-Performance Developer Portfolio
A professional, recruiter-focused portfolio designed for speed, accessibility, and clean UI/UX. Built with a "content-first" approach to ensure your skills and projects are the main focus.

💎 Design Philosophy
Recruiters spend an average of 6 seconds on a resume; the same applies to portfolios. This site was engineered to:

Load Instantly: Zero heavy frameworks or bloated libraries.

Reduce Friction: Clear navigation and high-contrast typography for easy scanning.

Demonstrate Attention to Detail: Subtle micro-interactions and smooth transitions that don’t distract from the content.

✨ Key Features
🌙 Adaptive Theming
Includes a seamless Light/Dark mode toggle. The system automatically detects user preferences and persists the choice across sessions using localStorage.

📱 Fully Responsive
Fluid layouts designed with CSS Grid and Flexbox. Whether viewed on a 4K monitor or an iPhone, the experience remains consistent and professional.

♿ Accessibility (a11y) First
Semantic HTML: Proper use of <main>, <section>, <nav>, and <article>.

Keyboard Navigable: All interactive elements are accessible via keyboard.

Screen Reader Friendly: Uses ARIA labels and descriptive attributes where necessary.

🏎️ Performance Optimized
Vanilla Stack: Built with pure HTML, CSS, and JS for a near-zero bundle size.
